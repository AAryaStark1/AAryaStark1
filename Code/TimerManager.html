定时器实现

一. 定时器创建

使用系统调用 timerfd_create 创建一个定时器，返回文件描述符 fd;

const int fileDescriptor{timerfd_create(CLOCK_MONOTONIC, 0)};

设置起始时间和间隔时间，通过 timerfd_settime 系统调用传递参数：

constexpr itimerspec time{
    {1, 0},  // 起始时间
    {1, 0}   // 间隔时间
};
timerfd_settime(fileDescriptor, 0, &time, nullptr);

使用 read 系统调用读取 fd 到用户空间中，返回距离上次读取的时间过期了多少个周期：

read(fileDescriptor, &expirations, sizeof(expirations));

通过上述三个系统调用实现时间定时器的设置。

二. 时间轮算法管理客户端连接

时间轮算法讲解：

使用 std::array 和 std::unordered_map 实现时间轮：

std::array<std::unordered_map<int, unsigned long>, 61> wheel;
std::unordered_map<int, std::chrono::seconds> location;
  • wheel：每个槽使用 unordered_map 存储客户端文件描述符 fd 和过期时间的整数倍。
  • location：用于定位 fd 所放置的槽位置。

提供以下四个外部接口进行时间定时器管理：

1. 添加定时器 

auto Timer::add(const int fileDescriptor, const std::chrono::seconds seconds) -> void {
    const auto point{(seconds + this->now) % (this->wheel.size() - 1)};
    const auto ticks{(seconds + this->now) / (this->wheel.size() - 1)};

    this->wheel[point.count()].emplace(fileDescriptor, ticks);
    this->location.emplace(fileDescriptor, point);
}

考虑当前时间轮 now 对应的槽位进行元素的添加。

2. 更新定时器 

auto Timer::update(const int fileDescriptor, const std::chrono::seconds seconds) -> void {
    const auto point{(seconds + this->now) % (this->wheel.size() - 1)};
    const auto ticks{(seconds + this->now) / (this->wheel.size() - 1)};

    this->wheel[this->location.at(fileDescriptor).count()].erase(fileDescriptor);
    this->wheel[point.count()].emplace(fileDescriptor, ticks);
    this->location.at(fileDescriptor) = point;
}

删除对应的文件描述符，并进行更新。

3. 移除定时器 

auto Timer::remove(const int fileDescriptor) -> void {
    if (const auto result{this->location.find(fileDescriptor)}; result != this->location.cend()) {
        this->wheel[result->second.count()].erase(fileDescriptor);
        this->location.erase(result);
    }
}

移除对应的定时客户端项。

4. 清除过期定时器 

auto Timer::clearTimeout() -> std::vector<int> {
    std::vector<int> result;

    while (this->timeout > 0) {
        std::unordered_map<int, unsigned long> &wheelPoint{this->wheel[this->now.count()]};
        for (auto element{wheelPoint.begin()}; element != wheelPoint.end();) {
            if (element->second == 0) {
                result.emplace_back(element->first);
                this->location.erase(element->first);
                element = wheelPoint.erase(element);
            } else [[likely]] {
                --element->second;
                ++element;
            }
        }

        ++this->now;
        this->now %= static_cast<decltype(this->now)>(this->wheel.size());
        --this->timeout;
    }

    return result;
}

返回所有过期的定时器项，通过 timefd 的周期数决定时间轮前进的步长，遍历所有槽中的客户端，将元素添加到 vector 数组中便于后续处理。

三、与小根堆定时器进行比较

时间复杂度

时间轮算法

  • 插入（Add）：O(1)
  • 删除（Remove）：O(1)
  • 调度（Tick）：O(N)，其中 N 是当前槽中的任务数

小根堆

  • 插入（Add）：O(log N)
  • 删除（Remove）：O(log N)
  • 调度（Tick）：O(1)

空间复杂度

时间轮算法

  • 空间复杂度：O(M)，取决于时间轮的槽数

小根堆

  • 空间复杂度：O(N)，与定时器任务的数量成正比

实现复杂度

时间轮算法

  • 实现复杂度：中等
  • 需要处理不同精度和时间范围的定时器任务，设计合理的时间轮结构

小根堆

  • 实现复杂度：较低
  • 维护二叉堆结构，操作直观简单
