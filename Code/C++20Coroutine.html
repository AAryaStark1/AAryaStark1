本篇探究C++20的协程实现

一、有栈协程和无栈协程
C++20的协程是典型的无栈协程的实现，在创建一个协程时，编译器会在堆区分配一块内存区域用于存储协程帧，协程帧中保留了协程的状态信息，包括协程当前的挂起点信息，协程创建时传入的参数，协程内创建的临时变量，在调用co_return(默认)时，协程帧中的参数得到释放，依赖编译器的实现。

goroutine则是典型的有栈协程的实现，goroutine使用go关键字进行标识，在协程创建的过程中会为分配一块2kb大小的内存空间，如何协程的栈空间不足，则会进行过扩展，因为使用了线程切换的实现思路，协程会保留当前的执行上下文信息，无需使用关键字co_wait等进行挂起点的标识，协程之间可以进行任意的切换。btw，这可能也是协程会和线程进行类比的原因，这样的实现会为协程栈空间大小的选择造成很大的挑战。

C++20的协程使用了状态机的原理，本质上是一个可重入的函数，通过协程的状态可以跳转到不同的执行逻辑中，在协程对应的汇编码中，存在大量的flag标识和jump指令，实现了协程的跳转。

二、C++20的协程关键组件
C++20的协程主要包括三个部分，Task、Promise以及一个Awaiter可等待对象。
Task为协程的外部封装类，通常Task负责与外部函数进行逻辑计算，一般情况下Task对象中只会保留协程的句柄一个成员函数，通过设置Task对象的外部公用接口，调用coroutine_handle<> handle的内部实现，管理协程与外部逻辑的通信。

Promise为一个可承诺对象，负责管理协程的生命周期，协程在创建时首先会根据传入的参数创建一个promise对象。除此之外，promise中还能管理协程的需要保留的数据成员，并由外部的Task对象进行通信。Promise定义的协程的生命周期，一个Promise对象需要提供以下几个接口：
1、初始化挂起

constexpr auto initial_suspend() const noexcept { return std::suspend_always{}; }

返回的是一个可等待对象Awaiter，在C++20的协程中，每一个挂起点都是一个可等待的Awaiter对象，具体的实现细节再后续的章节，通常这里的initial_suspend初始化挂起只有两种选择suspend_always或是suspend_never，分别对应积极启动和消极启动的场景。一个典型的用法时使用suspend_never消极启动，再调用编译器提供的resume函数恢复协程的执行，再这之前可以进行promise内部对象的初始化。

constexpr auto final_suspend() const noexcept { return std::suspend_always{}; }

协程的最终挂起点，再协程执行co_return操作后会执行，负责协程的最终挂起，同initial_suspend一样，语言提供了suspend_always和suspend_never的两种实现，在suspend_never的场景下，协程会立即销毁等效于调用coroutine_handle的destory函数，销毁协程对象，使得协程句柄handle变为无效值，这个时候如果外部的Task对象调用了提供的destory接口，就会因为两次销毁导致协程句柄的悬空引用，造成访问到了无效的内存空间，导致进程崩溃。而在suspend_always则是比较通用的实现，最终挂起协程，由外部的Task对象负责管理其生命周期，在析构函数中调用handle的destory函数可以实现协程的RAII管理，配合上智能指针可以实现在不同代码逻辑的传递，同时使用suspend_always可以在执行完毕之后，有充分的时间从promise对象中返回得到的异步执行的结果。

eg.协程中coroutine_handle<promise_type>提供了done()数据成员，用于判断协程是否执行完成

auto Task::promise_type::get_return_object() -> Task {
    return Task{std::coroutine_handle<promise_type>::from_promise(*this)};
}

promise的返回对象，通过调用编译器提供的from_promise接口实现了从promise_type对象向coroutine_handle<promise_type>对象的转变，coroutine_handle<>是一个模板类，promise_type可以是用户自定义的promise类型，然后使用Task的默认构造函数实现了协程外部对象Task的构建，在协程帧创建之后则会调用该函数实现外部Task的构建。

如果需要调用co_yield、co_return则需要实现以下几个接口。

auto yield_value(int ret)
{
    cout << "Promise: yield_value called with value " << ret << "." << endl;
    mRetValue = ret;
    return std::suspend_always();
}

调用了yield_value(co_yield)的协程也可以成为生成器，将创建好的对象保留在promise的数据成员mRetValue中，同时将协程挂起，由外部的task调用相关的实现与外部的逻辑进行通信。

void return_void()
{
    cout << "Promise: return_void called." << endl;
    mRetValue = 20;
}

void return_value(int val)
{
    cout << "Promise: return_void called." << endl;
    mRetValue = val;
}

上述分别对应了协程的最终返回值co_return和co_return val的实现，将最终的结果保留到协程的数据成员mRetValue中。

3、Awaiter对象
一个Awaiter对象需要实现以下三个接口

constexpr auto await_ready() const noexcept { return false; }

await_ready返回一个bool值，用于判断当前协程的状态是否需要挂起。

auto await_suspend(std::coroutine_handle<Task::promise_type> handle)

handle参数为当前执行的协程句柄，典型的做法是保留到Awaiter的数据成员的coroutine_handle<promise_type>中，便于下次唤醒对应的协程，或是将Awaiter对象中的参数保留到promise中，实现协程的传递。await_suspend的返回值可以是一个bool值、void和另一个协程句柄。bool值、void都用于标识当前的协程是否需要继续执行，返回coroutine_handle等效于执行协程的resume函数恢复，不在函数体中执行可以有效避免协程反复递归造成的栈溢出问题。

auto await_resume() const

协程调用resume恢复执行后，会首先从挂起点执行await_resume()，这里如果需要通过这里的Awaiter返回对应的结果与外部的coroutine实现通信，返回异步执行的结果。

二、以下通过一个示例解决协程的执行流程的问题

#include <coroutine>
#include <iostream>

using namespace std;

struct RepeatAwaiter // awaiter (raw pointer) / awaitable (operator->)
{
    bool await_ready() const noexcept
    {
        cout << "RepeatAwaiter: await_ready called." << endl;
        return false;
    }

    std::coroutine_handle<> await_suspend(
        std::coroutine_handle<> coroutine) const noexcept
    {
        if (coroutine.done())
        {
            cout << "RepeatAwaiter: coroutine is done, returning noop_coroutine." << endl;
            return std::noop_coroutine();
        }
        else
        {
            cout << "RepeatAwaiter: coroutine is not done, returning the same coroutine." << endl;
            return coroutine;
        }
    }

    void await_resume() const noexcept
    {
        cout << "RepeatAwaiter: resuming coroutine." << endl;
    }
};

struct RepeatAwaitable // awaitable (operator->)
{
    RepeatAwaiter operator co_await()
    {
        cout << "RepeatAwaitable: creating RepeatAwaiter." << endl;
        return RepeatAwaiter();
    }
};

struct PreviousAwaiter
{
    std::coroutine_handle<> mPrevious;

    bool await_ready() const noexcept
    {
        cout << "PreviousAwaiter: await_ready called." << endl;
        return false;
    }

    std::coroutine_handle<> await_suspend(
        std::coroutine_handle<> coroutine) const noexcept
    {
        if (mPrevious)
        {
            cout << "PreviousAwaiter: Resuming previous coroutine." << endl;
            return mPrevious;
        }
        else
        {
            cout << "PreviousAwaiter: No previous coroutine, returning noop_coroutine." << endl;
            return std::noop_coroutine();
        }
    }

    void await_resume() const noexcept
    {
        cout << "PreviousAwaiter: Resuming coroutine after previous coroutine." << endl;
    }
};

struct Promise
{
    auto initial_suspend()
    {
        cout << "Promise: initial_suspend called." << endl;
        return std::suspend_always();
    }

    auto final_suspend() noexcept
    {
        cout << "Promise: final_suspend called." << endl;
        return PreviousAwaiter{mPrevious};
    }

    void unhandled_exception()
    {
        cout << "Promise: unhandled_exception called." << endl;
        throw;
    }

    auto yield_value(int ret)
    {
        cout << "Promise: yield_value called with value " << ret << "." << endl;
        mRetValue = ret;
        return std::suspend_always();
    }

    void return_void()
    {
        cout << "Promise: return_void called." << endl;
        mRetValue = 20;
    }

    std::coroutine_handle<Promise> get_return_object()
    {
        cout << "Promise: get_return_object called." << endl;
        return std::coroutine_handle<Promise>::from_promise(*this);
    }

    int mRetValue = 0;
    std::coroutine_handle<> mPrevious = nullptr;
};

struct Task
{
    using promise_type = Promise;

    Task(std::coroutine_handle<promise_type> coroutine) : mCoroutine_(coroutine)
    {
        cout << "Task: Task created." << endl;
    }

    Task(Task &&) = delete;

    ~Task()
    {
        cout << "Task: Task destroyed." << endl;
        mCoroutine_.destroy();
    }

    std::coroutine_handle<promise_type> mCoroutine_;
};

struct WorldTask
{
    using promise_type = Promise;

    WorldTask(std::coroutine_handle<promise_type> coroutine)
        : mCoroutine_(coroutine)
    {
        cout << "WorldTask: WorldTask created." << endl;
    }

    WorldTask(WorldTask &&) = delete;

    ~WorldTask()
    {
        cout << "WorldTask: WorldTask destroyed." << endl;
        mCoroutine_.destroy();
    }

    struct WorldAwaiter
    {
        bool await_ready() const noexcept
        {
            cout << "WorldAwaiter: await_ready called." << endl;
            return false;
        }

        std::coroutine_handle<> await_suspend(
            std::coroutine_handle<> coroutine) const noexcept
        {
            cout << "WorldAwaiter: await_suspend called." << endl;
            mCoroutine_.promise().mPrevious = coroutine;
            return mCoroutine_;
        }

        void await_resume() const noexcept
        {
            cout << "WorldAwaiter: await_resume called." << endl;
        }

        std::coroutine_handle<promise_type> mCoroutine_;
    };

    auto operator co_await()
    {
        cout << "WorldTask: operator co_await called." << endl;
        return WorldAwaiter{mCoroutine_};
    }

    std::coroutine_handle<promise_type> mCoroutine_;
};

WorldTask world()
{
    cout << "world: starting execution." << endl;
    co_yield 422;
    cout << "world: yielded 422." << endl;
    co_yield 444;
    cout << "world: yielded 444." << endl;
    co_return;
}

Task hello()
{
    cout << "hello: constructing worldTask." << endl;
    WorldTask worldTask = world();
    cout << "hello: constructed worldTask, waiting for world." << endl;
    co_await worldTask;
    cout << "hello: received return from world " << worldTask.mCoroutine_.promise().mRetValue << "." << endl;
    co_await worldTask;
    cout << "hello: received return from world " << worldTask.mCoroutine_.promise().mRetValue << "." << endl;
    cout << "hello: yielding 42." << endl;
    co_yield 42;
    cout << "hello: yielding 12." << endl;
    co_yield 12;
    cout << "hello: yielding 6." << endl;
    co_yield 6;
    cout << "hello: ends execution." << endl;
    co_return;
}

int main()
{
    cout << "main: starting hello coroutine." << endl;
    Task t = hello();
    cout << "main: created hello task." << endl; // Task created, coroutine not started
    while (!t.mCoroutine_.done())
    {
        t.mCoroutine_.resume();
        cout << "main: received hello result " << t.mCoroutine_.promise().mRetValue << "." << endl;
    }
    cout << "main: hello coroutine completed." << endl;
    return 0;
}
